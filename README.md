## Setup

Clone the project

```bash
  git clone https://github.com/RevenBot/WordleSolver
```

Go to the project directory

```bash
  cd WordleSolver
```

Install dependencies

```bash
  dotnet workload install maui
```

Build the app

```bash
  dotnet build WordleSolver.csproj -f|--framework <FRAMEWORK>
```
